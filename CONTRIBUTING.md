# Translations

- Fork the repository.
- Open the `lang` directory.

## Adding a new language

- Find your language ISO code (Google it) and create `your_language_code.json`.
- Copy the translations from another language file.
- Translate all strings into your language.
- Commit with a message like: "Add [language name] translations".

## Modifying an existing language

- Open the translation file you want to edit.
- Make your changes and add your name to the "authors" list.
- Commit with a message like: "Modify [language name] translations".

# Other Contributions

If you're fixing bugs or adding new features, please:
- Use verbs in present tense in your commit messages.
- Avoid emojis or unusual characters in commit messages.
